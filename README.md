## Getting Started

First, run the development server:

Чтобы запустить фронт из корня папки в терминале нужно набрать

```bash
npm i
```

and

```bash
npm run dev
```

Чтобы запустить бэк из корня папки server в терминале нужно набрать

```bash
npm i
```

and

```bash
npm run start
```
